Installation

After installation trough DirectAdmin no data is showing up, this is normal.
The script 'add.sh' should be called preferably by crontab to work, this can
be done by running 'install_cron.sh' in '/usr/local/directadmin/plugins/load_monitor/'
(it should keep current crontab intact, but do it at your own risk!) or
by adding it manually to crontab using 'crontab -e':

*/5 * * * * cd /usr/local/directadmin/plugins/load_monitor/exec/; ./add.sh;

The '5' in the command is the interval in minutes the script should run in (5 by default),
change this if you like but be aware your machine might slow down if you call the script too often.

Make sure the exec is added under the root user or otherwise it might not collect all data you want.
